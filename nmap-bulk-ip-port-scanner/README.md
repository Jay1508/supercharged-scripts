## Usage ##

**nmap-bulk-ip-port-scanner (PHP Script)**: a php script to scan a list of ip addresses against a set of ports. The script logs ip addresses that have one of the specified list of ports open. Useful for network scanning and detection of specific types of servers (printers, IP cameras, etc). 

> Don't forget to run
> `composer update` from the root of the nmap-bulk-ip-port-scanner directory to load dependencies.
